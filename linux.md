# Linux Command Line Cheat-Sheet

## Simple Delete a directory and all its content
```
rm -r <directory name>
```

## Show what ports are running (useful for troubleshooting
```
sudi netstat -at | less
```

## Change volumes from Bash command line
```
# Change to F: drive
cd /f/

#Query volumes
cd /volumes
```
## Unpack Tar file
```
# Unpack
tar -xvf file.tar
```
